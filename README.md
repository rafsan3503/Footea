# Footea

A Professional photography review service

# Live Site Link

Open website: https://footeo-d76c1.web.app

## Features & Funtionality

- Animated header,navbar login and sign up page
- Responsive navbar
- Beautiful footer
- Home Section with service and gallery
- user log in status
- Firebase Authentication
- Google, Twitter, Github and Email log in
- Forget password link
- Private Routes for safety
- Blog Sections
- Add Service Section
- My reviews
- Error Page

## Technologies

- Firebase
- React Router Dom
- React Toastify
- Tailwind
- Hyper Ui
- Mamba UI
- React image view
- Kit Wind
- React Icon
- Sweet Alert
- JWT token
- MongoDb
